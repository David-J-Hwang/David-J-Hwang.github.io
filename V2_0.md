# David-J-Hwang.github.io - v2.0

## 프로젝트 목표
- 기존에 React로 만들었던 개인 홈페이지를, Next.js 프레임워크를 포함한, 디스플레이 크기별 반응형 디자인을 적용한 v2 버전으로 개량한다.
- Next.js를 적용하면서 Next.js 프레임워크에 대한 이해를 높이고, 좀 더 나은 디자인을 적용해 UI/UX를 개선한다.
- 기존에 가지고 있던 라우트들은 그대로 유지하되, 각각의 라우트들에 대한 개발작업을 진행한다.(블로그, 포트폴리오 프로젝트)


## 세부 개발작업
- 기존의 React 프로젝트를 Next.js 프로젝트로 변환한다.
- 디스플레이 크기에 따른 반응형디자인을 적용한다.
- 라이트모드/다크모드를 지원한다.(토글버튼은 상단바에 넣기)
- 페이지의 주요색상은 초록색으로 정하고, 라이트모드에서는 약간 밝은 초록색, 다크모드에서는 어두운 초록색을 적용한다.
- 홈페이지의 링크 버튼들은 각 서비스의 대표색상을 바탕색으로 적용한다.
- 우선 홈페이지를 개선하는 작업을 우선 진행하고, 완료되면 `/projects` 라우트와 `/blog` 라우트 작업을 진행한다.

## 세부 개발작업 - /projects

- `/projects` 라우트에는 이번에 새로 작업한 `/blackjack`과 `/lucky-wheel` 프로젝트 링크를 추가한다.
- `/projects` 라우트에 있는, 이 프로젝트에 포함되어 있는 코드들은 일단 제거하고, 각각의 프로젝트를 이 프로젝트의 코드에 넣는 대신 각 프로젝트 페이지의 링크로 보내도록 한다.
- 폰트는 Noto Sans KR을 일단 적용하되, 작업하면서 더 괜찮은 폰트를 발견하면 그걸 적용한다.

## 세부 개발작업 - /blog
- 우선 홈페이지 및 라우트 작업에 집중하고, /projects 라우트 개발작업을 완료하는 경우 더 상세히 개발방향을 정한다.
